# Prosperity-Prognosticator-startup-success-prediction

Prosperity Prognosticator is a Machine Learning project that predicts startup success using key features like funding, team experience, and market size. Built using Python and Logistic Regression, the system performs data analysis, model training, and classification to help investors, entrepreneurs, and policymakers make data-driven decisions.
